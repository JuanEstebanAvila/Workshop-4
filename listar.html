Listar todas las actividades

Endpoint GET /api/actividades — Sin parámetros de entrada. La tabla muestra el List<ActividadResponse> retornado por el backend.

GET

Entrada — Sin parámetros

Este endpoint no requiere ningún cuerpo ni parámetro. Solo se invoca la URL GET /api/actividades y el backend devuelve todas las actividades almacenadas en la base de datos H2.

URL del endpoint:

GET http://localhost:8080/api/actividades
+ Nueva actividad
200 OK

Respuesta — Body: List<ActividadResponse>

El backend retorna un arreglo JSON con todos los registros. Cada fila de la tabla corresponde a un objeto ActividadResponse.

idtitulotipoActividad fechaIniciofechaTerminacion idQuehaceridTutoridHijo
Pulsa "Consultar todas" para cargar los datos del backend.
