Modificar una actividad

Endpoint PUT /api/actividades/{id} — La entrada es el id en la URL más un ActividadDTO con todos los datos en el body. La tabla muestra el ActividadResponse modificado.

PUT

Entrada — Path: {id} + Body: ActividadDTO

Paso 1: ingresa el id de la actividad para cargar sus datos actuales. Paso 2: edita los campos que necesites y guarda. El endpoint recibe el id por URL y el objeto completo como JSON.
