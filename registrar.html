Endpoint POST /api/actividades — El formulario representa el ActividadDTO que se envía al backend. La tabla inferior muestra el ActividadResponse que devuelve.

El campo id no se incluye porque la base de datos H2 lo asigna automáticamente con @GeneratedValue(IDENTITY).
